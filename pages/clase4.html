Clase 4

CSS avanzado

  • Box modeling - Margin y Padding
    • Diferencia Margin y Padding
      • Margin - Espacio en blanco del border para afuera
      • Padding - Espacio en blanco entre el border y el contenido
    • 4 Formas de definirlos
      • margin: 10px - Todos los lados a la vez
      • margin: 10px 20px - Eje vertical y horizontal
      • margin: 10px 20px 30px 40px - Cada lado x separado
      • margin-top / margin-bottom / margin-left / margin-right - Cada lado individualmente
  • Contenedores Límite
    • Para qué se usa
    • Creación de un contenedor límite
      • Encerrar en un contenedor (div, main) lo que queremos que tenga contenedor límite
      • A ese contenedor, ponerle una class "cLimite"
      • En CSS, darle las siguientes reglas a esa clase
        • max-width: 50% - Defino ancho máximo del contenido
        • margin: 0 auto - Centrar el contenido
  • Propiedad Position
    • static - valor por defecto - flujo natural de HTML
    • relative - modificar la posicion de mi elemento partiendo de su posicion original en el HTML
    • absolute - modificar la posicion de mi elemento ignorando su posicion original en el HTML
    • fixed - fijaba la posición de mi elemento y no era afectado por el scroll
    • sticky - Relative + Fixed - cuando llega a x distancia, se "pega" a la pantalla
  • Propiedad z-index
    • z-index: (más abajo) -2, -1, 0, 1, 2 (más arriba)
    • Establecer el orden en objetos que se superponen
