Clase 3

Introducción a CSS

  • Qué es CSS, para que sirve
  • Sintaxis básica
  • Aplicar CSS a mi HTML
    • Asegurarme de estar trabajando en una carpeta y que ahí esté mi index.html
    • Crear un archivo "style.css" dentro de la carpeta de mi proyecto
    • Para vincular el HTML con el CSS, tengo que agregar, en el head de mi HTML, una etiqueta link
    • Creo una regla sencilla en CSS para probar si realmente funciona
  • Propiedades CSS básicas
    • color: color del texto
    • background-color: color de fondo de la etiqueta
    • text-align: alinear el texto (centrar, izq, der, just)
  • Selectores
    • Selector Universal (*) - Más general / liviano
    • Selector de Etiqueta (p)
    • Clases (.nombre_de_mi_clase)
    • ID (#nombre_de_mi_id) - Selector más específico / pesado
    • Palabra Reservada !important - Excepción, rompe la lógica de la cascada CSS
